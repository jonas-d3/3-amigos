# 3 Amigos Check-in (Quick Guide)

## 🎯 Formål

Afklare:

> **Er item klar til næste step — hvis ikke, hvad mangler?**

---

## ⏱ Setup

- 45 min max
- 3–5 PBI’er
- UX + UI + Dev

---

## 🔁 Flow (per PBI)

1. **Hvor er vi?**
   - UX / UI1 / UI2 / Dev

2. **Er det klar til næste step?**
   - Ja → videre
   - Nej → hvad mangler?

3. **Beslut output**
   - ✅ Klar til næste fase
   - ❌ Ikke klar → næste skridt + ansvarlig

---

## 🔄 Faser (kort)

### UX

- Flow giver mening
- Dev forstår det

### UI Iteration 1

- Retning er ok
- Ingen blockers

### UI Iteration 2

- Klar til dev
- Ingen åbne spørgsmål

### Development

- Matcher UX + UI
- Ingen kritiske issues

---

## ⚠️ Husk

- Samtale > proces
- Godt nok > perfekt
- Ingen rykker videre uden fælles forståelse
